# base_express_js

- download nodejs::
https://nodejs.org/en/download/

- install expressJS::
npm i express --save

- install expressJS generator::
npm i -g express-generator

- initalize ExpressJS::
express --view=ejs myapp

- go to directory & install::
cd myapp && install

- run it::
set DEBUG=myapp:* & npm start

- or using nodemon
npm i -g nodemon

-run using nodemon
nodemon
